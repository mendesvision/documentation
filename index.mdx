## Começando

Escolha por onde deseja começar sua jornada com a Eyo Wallet.

<Steps>
  <Step title="Crie sua conta">
    Acesse [eyowallet.com](https://eyowallet.com) e crie sua conta gratuitamente.
  </Step>
  <Step title="Obtenha sua API Key">
    Siga o tutorial [Obter API Key](/tutorials/get-api-key) para configurar suas credenciais.
  </Step>
  <Step title="Integre e automatize">
    Use nossa API ou conecte ao [Eyo Bot](/tutorials/integrar-eyo-bot) para automatizar suas vendas.
  </Step>
</Steps>
