---

O **Eyo Bot** é um bot de Discord que permite automatizar seu servidor com sistemas de vendas, moderação, verificação de usuários e muito mais. Neste tutorial, você aprenderá a conectar sua conta Eyo Wallet ao Eyo Bot.

<Tip>
  Ainda não tem um Eyo Bot? Adquira o seu em [eyoapplications.com](https://eyoapplications.com)
</Tip>

<Warning>
  **Limites de valor para pagamentos:**
  
  - **Valor mínimo**: R$ 1,00
  - **Valor máximo**: R$ 1.000,00
  
  Pagamentos ou produtos com valores fora desse intervalo resultarão em erro.
</Warning>

---

## Passo 1: Criar uma conta na Eyo Wallet

Acesse [eyowallet.com](https://eyowallet.com) e crie sua conta caso ainda não tenha uma.

---

## Passo 2: Acessar as Credenciais

Acesse a página de credenciais da sua conta:

<Card title="Credenciais" icon="key" href="https://eyowallet.com/dashboard/credentials">
  https://eyowallet.com/dashboard/credentials
</Card>

---

## Passo 3: Autorizar todos os IPs

Na página de credenciais, autorize todos os IPs para permitir que o Eyo Bot se conecte à sua conta.

<Frame caption="Autorizar todos os IPs">
  <img src="/images/tutoriais/autorizar-ips.png" alt="Tela mostrando como autorizar todos os IPs" />
</Frame>

<Warning>
  Ao autorizar todos os IPs, qualquer serviço com sua API Key poderá acessar sua conta. Mantenha sua chave em segurança.
</Warning>

---

## Passo 4: Copiar a API Key

Na mesma página, copie a API Key que já está criada.

<Frame caption="Copiar API Key">
  <img src="/images/tutoriais/copiar-api-key.png" alt="Tela mostrando onde copiar a API Key" />
</Frame>

<Tip>
  Clique no ícone de copiar ao lado da chave para copiá-la automaticamente.
</Tip>

---

## Passo 5: Preencher no Eyo Bot

No Discord, use o comando /painel e navegue para Configurações > Formas de Pagamento > PIX > Eyo Wallet e cole sua API Key.

<Frame caption="Preencher API Key no Bot">
  <img src="/images/tutoriais/preencher-bot.png" alt="Tela do Eyo Bot para preencher a API Key" />
</Frame>

---

## Pronto!

Sua integração está configurada. Agora o Eyo Bot pode processar pagamentos usando sua conta Eyo Wallet.

<Tip>
  Precisa de ajuda? Junte-se ao nosso [Discord](https://t.me/eyowallet) para suporte.
</Tip>
